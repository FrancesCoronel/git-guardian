Login to Twitter and show top 3 headlines from when you signed up.

## Installation

Use the following command to serve up locally.

```shell
#
$ npm i -g firebase-tools # Install firebase-tools if you haven't already
$ firebase serve   # Start development server
```

CodeKit 3 was used to compile assets.

## Usage

3 Endpoints

- Twitter
    + OAuth
    + User's tweet
    + User's account creation date
- NYTimes Articles: https://api.nytimes.com/svc/search/v2/articlesearch.json
    + article.headline.main
    + article.lead_paragraph
    + article.pub_date
    + article.web_url
